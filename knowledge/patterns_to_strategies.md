## СИЛЬНЕЙШИЙ БЫЧИЙ СИГНАЛ:
```
1. Цена в восходящем тренде (EMA(50)>EMA(200))
2. ADX > 25
3. Hammer или Bullish Engulfing
4. RSI > 40 и растёт
5. Объём выше среднего
= Полная позиция (риск 2%)
```

## СИЛЬНЕЙШИЙ МЕДВЕЖИЙ СИГНАЛ:
```
1. Цена в нисходящем тренде
2. ADX > 25
3. Shooting Star или Bearish Engulfing
4. RSI < 60 и падает
5. Объём выше среднего
= Полная позиция (риск 2%)
```

## СИЛЬНЫЙ СИГНАЛ НА ОТКАТ:
```
1. Тренд подтверждён (EMA crossover)
2. ADX > 25
3. Цена возвращается к EMA(21)
4. RSI < 45 (бычий) / > 55 (медвежий)
5. Свечной паттерн разворота (Hammer, Engulfing)
= Половина позиции
```

## СИГНАЛ НА ПРОБОЕ:
```
1. Цена в диапазоне/консолидации
2. Bollinger Bands сужаются
3. Объём растёт при сужении
4. Пробой выше High / ниже Low
5. Закрытие за пределами уровня
= Вход после подтверждения (retrace к уровню)
```

## СИГНАЛ НА ДИВЕРГЕНЦИИ:
```
1. Цена делает HH (Higher High)
2. RSI делает LH (Lower High)
3. Дополнительно: объём падает
= Приготовиться к шорту
= Вход когда RSI пробьёт свой последний Low
```

## 5m-15m (Scalping/Intraday):
```
✓ Объём > 1.5x среднего
✓ Нет важных новостей (NON-FARM, FOMC)
✓ Сессия: 8-16 MSK оптимально
✓ Spread < 0.1% (для акций/крипты)
✓ Тренд на старшем таймфрейме подтверждает
```

## 1H-4H (Swing):
```
✓ ADX > 20
✓ Объём нормальный
✓ Нет новостей 2+ часа после входа
✓ Риск/Прибыль >= 1:2
✓ Не более 3 позиций одновременно
```

## 1D+ (Position):
```
✓ Тренд на недельном подтверждён
✓ ADX > 25
✓ Новостной фон стабильный
✓ Риск/Прибыль >= 1:2
✓ Stop не ближе 3 ATR
```

═══════════════════════════════════════════════════════════
ANTI-PATTERNS (КОГДА НЕ ТОРГОВАТЬ)
═══════════════════════════════════════════════════════════

1. DOJI после сильного тренда = осторожно (может быть просто паузой)
2. HEAD & SHOULDERS с очень узкими плечами = низкая надёжность
3. BREAKOUT с объёмом < 1.2x среднего = высокая вероятность ложного
4. DOUBLE TOP на недельном с расхождением более 5% между пиками = ненадёжно
5. RSI > 90 или < 10 = экстремально, часто ловушка
6. Паттерн против основного тренда = игнорировать без сильного подтверждения
7. GAP UP после 3+ дней роста = разворотный паттерн (exhaustion gap)
8. Паттерн на низкой ликвидности = slippage и ложные сигналы
