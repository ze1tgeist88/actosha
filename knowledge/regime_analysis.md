MARKET REGIMES (РЫНОЧНЫЕ РЕЖИМЫ)
═══════════════════════════════════════════════════════════

## Определение режима

Режим определяется по:
1. Наклон EMA(200) — направление долгосрочного тренда
2. ADX — сила тренда
3. Волатильность (ATR relative to price)
4. Ценовое действие (Higher Highs vs Range)

## 4 Основных Режима

### 1. STRONG TREND (Сильной тренд)
Признаки:
  - EMA(50) > EMA(200) = бычий, EMA(50) < EMA(200) = медвежий
  - ADX > 25 (растущий)
  - Цена делает Higher Highs (бычий) / Lower Lows (медвежий)
  - Минимум откатов 20-38%

Стратегия:
  - Торговать В НАПРАВЛЕНИИ тренда
  - Использовать EMA для входов на откатах
  - RSI/CCI для перекупленности игнорировать в направлении тренда
  - Stop-loss за последним Low/High

### 2. WEAK TREND (Слабый тренд)
Признаки:
  - ADX 15-25
  - EMA(50) и EMA(200) близко друг к другу
  - Цена делает небольшие HH/HL или LH/LL

Стратегия:
  - Ограничить торговлю
  - Использовать Range-стратегии
  - Торговать от границ диапазона

### 3. RANGE (Боковой рынок)
Признаки:
  - Цена между горизонтальными уровнями
  - ADX < 20
  - Нет новых High/Low
  -几次 тестирование уровней

Стратегия:
  - Покупать у поддержки, продавать у сопротивления
  - RSI(14) < 30 = покупка, > 70 = продажа
  - ИЗБЕГАТЬ трендовых индикаторов
  - Использовать Bollinger Bands (касание полос = сигнал)

### 4. VOLATILE/CHAOS (Высокая волатильность)
Признаки:
  - ATR резко растёт
  - Цена пробивает уровни без закрытия
  - Новостной фон

Стратегия:
  - Уменьшать позиции
  - Использовать широкие stop-loss
  - Избегать входа на новостях
  - Ждать стабилизации

STRATEGY CONSTRUCTION FRAMEWORK
═══════════════════════════════════════════════════════════

## Step 1: Определить режим рынка
```
IF ADX > 25 AND EMA(50) > EMA(200) → BULLISH_TREND
IF ADX > 25 AND EMA(50) < EMA(200) → BEARISH_TREND
IF ADX < 20 → RANGE
IF ATR > 2x Average → HIGH_VOLATILITY
```

## Step 2: Выбрать тип стратегии

### Trend-Following (для трендов):
- EMA Crossover (9/21)
- Breakout with ADX filter
- Trendline break

### Mean Reversion (для Range):
- Bollinger Bands
- RSI Extreme levels
- Support/Resistance bounce

### Momentum (для сильных трендов):
- RSI/CCI в направлении тренда
- MACD histogram direction
- Volume surge confirmation

## Step 3: Вход (Entry)
```
Правила входа:
1. Режим подтверждён (ADX, EMA)
2. Сигнал от индикатора (RSI/CCI/Stochastic)
3. Подтверждение объёмом (OBV/VWAP)
4. Риск/Прибыль > 1:2 минимум
```

## Step 4: Управление позицией

### Stop-Loss:
```
Trend: 1.5x ATR от входа или последний Low
Range: За уровнем поддержки/сопротивления
Momentum: При смене сигнала индикатора
```

### Take-Profit:
```
Фиксированный: 2-3x риска
Измерительный: Высота фигуры / ATR
Trailing: EMA(20) для краткосрочных, EMA(50) для среднесрочных
```

### Scale-Out:
```
Выход 50% позиции при 1:1 риск/прибыль
Выход 75% при 1:2
Оставшееся пустить с trailing stop
```

## Step 5: Фильтры

### Обязательные фильтры:
```
1. Режим рынка совместим со стратегией?
2. Объём подтверждает движение?
3. Новостной фон не противоречит?
4. Время суток (сессия)?
```

### Избегать:
```
- Вход против тренда без подтверждения
- Вход на низком объёме
- Вход во время новостей высокой важности
- Вход если ADX < 20 для трендовой стратегии
```

BACKTESTING PRINCIPLES
═══════════════════════════════════════════════════════════

## Minimum requirements для надёжного бэктеста:
1. Минимум 100 сделок
2. Данные минимум 2+ лет
3. Разные рыночные условия (тренд, range, волатильность)
4. Out-of-sample тестирование

## Metrics для оценки стратегии:
```
Win Rate: % прибыльных сделок
Profit Factor: Gross Profit / Gross Loss (>1.5 хорошо)
Sharpe Ratio: (Return - Risk-Free) / StdDev (>1.5 хорошо)
Max Drawdown: Максимальная просадка (<20% acceptable)
Recovery Factor: Total Net Profit / Max Drawdown (>2 хорошо)
Avg Trade: Средняя прибыль/убыток
 expectancy: (WinRate × AvgWin) - (LossRate × AvgLoss)
```

## Проверка на overfitting:
```
- Попробовать на других инструментах (BTC → ETH)
- Изменить таймфрейм
- Изменить параметры индикаторов на ±20%
- Протестировать на out-of-sample данных
Если результаты сильно отличаются → overfitting
```

═══════════════════════════════════════════════════════════
RISK MANAGEMENT
═══════════════════════════════════════════════════════════

## Position Sizing
```
Риск на сделку: 1-2% депозита максимум
Формула: Position Size = (Account × Risk%) / (Entry - Stop)
```

## Максимальные потери:
```
Дневной лимит: 3-5% депозита
Недельный лимит: 10%
Месячный лимит: 15-20%
```

## Правило:
```
1. Если дневной лимит достигнут → СТОП торговли
2. Если неделя в минусе 10% → анализ причин
3. Если месяц в минусе 20% → полный пересмотр системы
```

## Приоритет паттернов по режиму:
```
STRONG_UPTREND:
  1. Pullback к EMA (высокая вероятность)
  2. Breakout выше High
  3. Moving Average crossover
  4. Bull Flag
  5. ИЗБЕГАТЬ: Buy at resistance

STRONG_DOWNTREND:
  1. Rally к EMA (высокая вероятность)
  2. Breakdown ниже Low
  3. Moving Average crossover
  4. Bear Flag
  5. ИЗБЕГАТЬ: Sell at support

RANGE:
  1. Buy at support, sell at resistance
  2. Bollinger Bands touches
  3. RSI < 30 buy, > 70 sell
  4. ИЗБЕГАТЬ: Trend-following strategies
```

EXAMPLE STRATEGY: EMA + RSI + ATR
═══════════════════════════════════════════════════════════

Params:
  EMA_FAST = 9
  EMA_SLOW = 21
  RSI_PERIOD = 14
  ATR_PERIOD = 14
  ATR_MULT = 1.5

Entry Long:
  1. EMA(9) > EMA(21) (тренд вверх)
  2. ADX > 25 (тренд подтверждён)
  3. Цена откатилась к EMA(21)
  4. RSI(14) < 45 и разворачивается вверх (моментум восстанавливается)
  5. RSI(14) < 30 = сильнее сигнал (перепроданность)

Entry Short:
  1. EMA(9) < EMA(21) (тренд вниз)
  2. ADX > 25
  3. Цена откатилась к EMA(21)
  4. RSI(14) > 55 и разворачивается вниз
  5. RSI(14) > 70 = сильнее сигнал

Stop-Loss:
  1.5 × ATR от входа (фильтр волатильности)

Exit:
  - Take-Profit: 2x риска (R:R = 1:2)
  - ИЛИ RSI выходит в противоположную зону
  - ИЛИ EMA crossover в противоположном направлении

Timeframe: 1H-4H (оптимально для крипты)
Symbols: High liquidity (BTC, ETH)
Max simultaneous positions: 3
