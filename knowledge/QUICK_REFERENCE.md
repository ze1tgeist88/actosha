# Шпаргалка для ежедневной торговли

═══════════════════════════════════════════════════════════
ПЕРВЫЙ ШАГ: ОПРЕДЕЛИТЬ РЕЖИМ
═══════════════════════════════════════════════════════════

ADX < 20 → RANGE → Mean Reversion (Bollinger, RSI 30/70)
ADX 20-25 → СЛАБЫЙ ТРЕНД → Ограничить позиции
ADX > 25 + EMA(50)>EMA(200) → ВВЕРХ → Тренд вверх
ADX > 25 + EMA(50)<EMA(200) → ВНИЗ → Тренд вниз

═══════════════════════════════════════════════════════════
СИГНАЛЫ ВХОДА — БЫСТРАЯ ПРОВЕРКА
═══════════════════════════════════════════════════════════

LONG:
□ Цена > EMA(200)          — долгосрочный бычий
□ EMA(9) > EMA(21)         — краткосрочный бычий  
□ ADX > 20                 — тренд есть
□ RSI 35-50               — зона для входа
□ Объём подтверждает       — усиливает сигнал
= +1-2 если всё ДА

SHORT:
□ Цена < EMA(200)          — долгосрочный медвежий
□ EMA(9) < EMA(21)         — краткосрочный медвежий
□ ADX > 20                 — тренд есть
□ RSI 50-65               — зона для входа
□ Объём подтверждает       — усиливает сигнал
= +1-2 если всё ДА

═══════════════════════════════════════════════════════════
STOP-LOSS БЫСТРЫЙ РАСЧЁТ
═══════════════════════════════════════════════════════════

DEFAULT: 1.5 × ATR от входа
INTRADAY (5m-15m): 0.5-1% от входа
SWING (1H-4H): 1.5-2.5% от входа
POSITION (1D+): 3-5% от входа

═══════════════════════════════════════════════════════════
TARGET РАСЧЁТ
═══════════════════════════════════════════════════════════

МИНИМУМ: 2x риск (R:R = 1:2)
ХОРОШО: 3x риск (R:R = 1:3)
ВЗЯТЬ 50% при 1:1, остаток trailing

═══════════════════════════════════════════════════════════
ОБЪЁМ — БЫСТРАЯ ПРОВЕРКА
═══════════════════════════════════════════════════════════

Volume > 2x среднего на пробое = СИЛЬНЫЙ
Volume > 1.5x среднего = НОРМАЛЬНЫЙ
Volume < среднего = СЛАБЫЙ (избегать)
Volume растёт на откате = БЫЧИЙ
Volume падает на откате = ПОДОЗРИТЕЛЬНО

═══════════════════════════════════════════════════════════
RSI БЫСТРАЯ ИНТЕРПРЕТАЦИЯ
═══════════════════════════════════════════════════════════

RSI > 85 → ЭКСТРЕМАЛЬНО ВВЕРХ → НЕ ПОКУПАТЬ
RSI > 70 → ПЕРЕКУПЛЕНО → ГОТОВИТЬСЯ К ПРОДАЖЕ
RSI 50-70 → НЕЙТРАЛЬНО ВВЕРХ → можно держать
RSI 40-50 → НЕЙТРАЛЬНО ВНИЗ → осторожно
RSI 30-45 → ЗОНА ПОКУПКИ (при бычьем тренде)
RSI < 30 → ПЕРЕПРОДАНО → ГОТОВИТЬСЯ К ПОКУПКЕ
RSI < 15 → ЭКСТРЕМАЛЬНО ВНИЗ → НЕ ПРОДАВАТЬ

═══════════════════════════════════════════════════════════
ГОЛОСОВАНИЕ СИГНАЛОВ
═══════════════════════════════════════════════════════════

Сложи очки:
+1 Тренд в мою сторону (ADX>25 + EMA)
+1 Объём подтверждает
+1 RSI в благоприятной зоне
+1 Паттерн >70% надежности

4-5 очков = Полная позиция (2% риска)
2-3 очка = Половина позиции (1% риска)
0-1 очко = Пропустить

═══════════════════════════════════════════════════════════
РИСК-МЕНЕДЖЕНТ
═══════════════════════════════════════════════════════════

Риск на сделку: 2% МАКСИМУМ
Дневной лимит: 3% потерь
Макс. позиций: 3 одновременно
Цель дня: +1-2% = хорошо
Минимум R:R: 1:2

═══════════════════════════════════════════════════════════
DIFFERENT TIMEFRAMES
═══════════════════════════════════════════════════════════

5M-15M: Scalping, быстрые входы, маленькие цели
1H-4H: Swing, основной рабочий для intraday
1D: Position, свинг-трейдинг на 1-5 дней
1W+: Invest/Trading, месяцы

═══════════════════════════════════════════════════════════
ANTI (НЕ ДЕЛАТЬ)
═══════════════════════════════════════════════════════════

✗ Входить на новостях
✗ Входить против тренда без 4+ очков
✗ Держать позу через новости >High impact
✗ Увеличивать позицию в убытке (averaging down)
✗ Рисковать >2% на сделку
✗ Торговать на ADX < 15 трендовой стратегией
✗ Покупать RSI > 85 или продавать RSI < 15

═══════════════════════════════════════════════════════════
КОГДА ДЕЛАТЬ ПАУЗУ
═══════════════════════════════════════════════════════════

3 убытка подряд → пауза 1 час
5 убытков подряд → конец дня
Дневной лимит достигнут → СТОП
ADX < 15 на всех парах → нет сделок
Экстремальная волатильность (ATR x3) → reduced size
